Событие, при котором срабатывает перемещение линии

Дотсупна настройка нескольких вариантов работ:

click - перемещение линии происходит по клику;

motion-in - перемещение линии присходит при наведении на пункт;

motion-in-out - перемещение линии присходит при наведении на пункт и когда мышь уходит со всех пунктов, то линия становится на начальное положение

const magicThird = new MagicLine('[data-list-click]', {
   usedEvent: 'click',

Определение мобилки

Плагин может определить наличие тач экрана и перевести собтия наведения мыши на событие клика.

Также есть возможность удалить линию на опреденном разрешении экрана.

Все настройки могут быть указаны в инициализации или дата-атрибутом.

Дата-атрибуты имеют высший приоритет!

Перевод события из наведения в клик на экранах меньше 900px. 

const magicSeventh = new MagicLine('[data-list-motion-to-click]', {
   media: 'max,900',
   mediaType: 'motion-to-click',
});
